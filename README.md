# Text Compare Tool

두 개의 텍스트를 비교하고, 차이를 시각적으로 보여줍니다.
업로드한 파일 또는 직접 입력한 텍스트를 기준으로 문자/단어 단위의 일치율과 오류율을 계산하고, 
차이점을 하이라이트하여 직관적으로 확인할 수 있도록 합니다.
